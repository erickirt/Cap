Instant Mode is Cap's fastest recording-to-link workflow. It uploads recording data while the session is active, then completes processing after you stop.

## Record in Instant Mode

1. Open Cap Desktop and select **Instant**.
2. Choose a display, window, area, or camera-only target.
3. Select the camera, microphone, and supported audio options you want to include.
4. Click **Record** and complete the countdown.
5. Use the recording controls to pause, resume, restart, or stop.

If you prefer global controls, assign them first in **Settings > Shortcuts**. Cap does not ship default recording hotkeys.

## After you stop

Cap finalizes the upload and prepares the share page. Processing time depends on the recording and connection, so treat the link as ready only after Cap reports it.

The Cap appears in the desktop library and web dashboard, where you can copy its link, rename it, change supported access settings, organize it, or delete it.

## What viewers get

A public share link opens in a browser without requiring an account. Private and password-protected Caps enforce their configured access rules.

The viewer supports playback controls and the playback-speed options currently offered by Cap. Captions, transcripts, summaries, chapters, downloads, comments, and reactions appear only when the content has been produced and the Cap or organization settings allow them. Posting feedback requires sign-in.

## Storage

New Instant uploads use the active storage provider: Cap-managed storage, an individual or organization Google Drive integration, or configured S3-compatible storage. Changing a provider affects future uploads; it does not automatically move existing Caps.

See [Google Drive](/docs/teams/google-drive) or the [S3 guides](/docs/s3-config/aws-s3).

## When to use Instant Mode

Instant Mode fits bug reports, customer support, async updates, quick walkthroughs, and other cases where a shareable Cap matters more than editing first.

Use [Studio Mode](/docs/recording/studio-mode) when you want a local project, timeline editing, visual styling, or local export before sharing.
